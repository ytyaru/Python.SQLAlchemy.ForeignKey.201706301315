# このソフトウェアについて

SQLAlchemyを使ってみた。

SQLAlchemyでテーブル作成、SELECT,INSERT,UPDATE,DELETEをしてみた。

# 開発環境

* Linux Mint 17.3 MATE 32bit
* [Python 3.4.3](https://www.python.org/downloads/release/python-343/)
* [SQLite](https://www.sqlite.org/) 3.8.2
